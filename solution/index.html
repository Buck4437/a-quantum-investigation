This puzzle requires the solver to stabilise five crates using interference. The solver needs to match the waveform of the stable state, and match the inversion of the unstable state.

There are five modes for the device, representing different kinds of waveform:

  • 1: Sine
  • 2: Tangent
  • 3: Sawtooth
  • 4: Square
  • 5: Triangle

Note that there is a small margin for error. If the frequencies are close enough, then it would count as correct.

After solving the puzzle, the solver would obtain the answer COLLAPSE.

This puzzle tells the solver that Alice's and Bob's minds are entangled, suggesting some kind of relationship between them. If we press "Give Up", the puzzle will show the correct answer, which gives us information to deduce the entanglement relationship between Alice and Bob.

The relationships are as follows:

Level 1: Alice and Bob are thinking of the same thing.

Level 2: Alice's and Bob's number sums up to 100.

Level 3: Bob's letter is Alice's letter shifted 4 places (e.g. A -> E, B -> F... Z -> D).

Level 4: The first two bits of Alice's and Bob's bit string are equal. The last three bits of Alice's and Bob's bit string are flipped. (e.g. if Alice's bit string is "00000", then Bob's bit string is become "00111").

After solving the puzzle, the solver would obtain the answer TOGETHER.
